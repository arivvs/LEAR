eng version:

Lear is a script written in Python that allows you to monitor computer sound and screen without being on the same Wi-Fi network.
Installation is done via Tailscale; you must register and log in using the same account. To monitor the screen, you must open the website with the IP (https://100.xx.xx.xx) that is listed in Tailscale in the column for the computer.

ru version:
LEAR это скрипт написанный на python, благодаря которому можно отслеживать звук и экран компьютера, не находясь в вай-файн сети.
